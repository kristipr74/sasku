# sasku

## Saku andmebaasi/database API

You have to download node in your computer

- https://nodejs.org/en/download/

# Run project

- git clone https://github.com/kristipr74/sasku
- cd sasku
- npm install
- npm start
  Go to http://localhost:3000/ping

# HTTP request methods

- GET
- POST
- PATCH
- DELETE

### Endpoints:

- players
- games
- groups
- results
- rounds

### Respond codes

```
{
ok: 200,
created: 201,
noContent: 204,
badRequest: 400,
notAuthorized: 401,
notFound: 404,
}
```
